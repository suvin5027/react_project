# React 게시판 연습 프로젝트

React를 연습하기 위한 게시판 웹 페이지입니다! 🚀

## 프로젝트 소개

- **목적**: React 학습 및 연습
- **기능**: 게시판 화면 구현 (목록, 상세, 작성 등)
- **특징**: 프론트엔드 화면만 구현 (백엔드 연동 X)

## 기술 스택

- **React** - UI 라이브러리
- **Vite** - 빠른 개발 서버와 빌드 도구
- **CSS** - 스타일링

## 접속 주소

- **개발 서버**: http://localhost:5173

## 개발 환경

- **Java** 21
- **Node.js** v20.20.2
- **npm** 10.8.2

## 시작하기

```bash
# 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```
